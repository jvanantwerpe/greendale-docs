=================
Request a feature
=================

..  toctree::
    :maxdepth: 1

This page details how you can give us requests regarding features for the Identity and Access Management solution.

Rules
=====

1. Requests should be done in writing.
2. Submit your request electronically.
3. The request should be phrased like a user story. 
   Something along the line as "As Developer, I want..., so that ..."
4. Please make sure that if legacy systems are affected...
   We won't accept improvements only bug fixes. Sorry!
5. Be around for questions regarding your submission - in case there a questions.

Submission
==========
Send us an E-Mail to team-greendale@zalando.de
